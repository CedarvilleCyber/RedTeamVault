Snaffler searches available network shares for credential information
```powershell
.\Snaffler.exe -c All -o output -v data
```

`-s`: print results to the console
`-o`: save to a file
`-v`: verbosity level
`-c`: specify domain controller

## Color codes: 
- Green: Low 
- Yellow: Medium
- Red: High severity
- ==Black==: Critical severity
